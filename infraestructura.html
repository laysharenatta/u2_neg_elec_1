Infraestructura del Proyecto

Estas son las especificaciones técnicas y decisiones de diseño que garantizan el correcto funcionamiento del sitio web de HoneyPop.

Procesador

2 vCPU — Permite manejar múltiples solicitudes simultáneas sin afectar el rendimiento del servidor.

Memoria RAM

4 GB RAM — Asegura fluidez en la ejecución del servidor web y la base de datos.

Almacenamiento

50 GB SSD — Ofrece una alta velocidad de lectura/escritura para mejorar la carga de datos.

Transferencia

1 TB mensual — Suficiente para soportar el tráfico esperado del sitio y las descargas de imágenes.

Sistema Operativo

Ubuntu Server 22.04 LTS — Estable, seguro y ampliamente compatible con servidores web.

Servidor Web y Base de Datos

Apache o Nginx con MySQL/MariaDB — Tecnologías confiables para aplicaciones web de alto rendimiento.

Conclusión

La infraestructura propuesta equilibra rendimiento, costo y escalabilidad, ideal para un negocio local en crecimiento como HoneyPop. Gracias al uso de tecnologías modernas y recursos adecuados, se garantiza una experiencia rápida, estable y segura para los usuarios y administradores del sistema.
